# 快捷键说明

## 默认快捷键

| 快捷键 | 操作 | 说明 |
|---|---|---|
| `Ctrl/Cmd + C`（复制时） | **自动翻译** | 复制即翻译，弹窗自动出现（**PDF 友好**） |
| `Ctrl/Cmd + Shift + T` | 翻译当前选中文本 | 弹窗显示结果 |
| `Ctrl/Cmd + P` | 打开命令面板 | 输入「Translate」搜索命令 |
| `Esc` | 关闭翻译弹窗 | 在弹窗打开时按 |
| 点击外部 / 拖动标题栏 | 关闭 / 拖动弹窗 | 弹窗打开时 |

---

## ⭐ 推荐用法：复制即翻译

适用所有场景（**包括 PDF**）：

```
1. 选中文字（编辑器 / PDF / 任何地方）
2. Ctrl+C 复制
3. 弹窗自动出现 ✅
```

可在设置中关闭「复制后自动翻译」。

---

## 自定义快捷键

Obsidian → Settings → Hotkeys → 搜「Scientific Translator」

可以重新绑定：
- `Scientific Translator: Translate selection`：翻译当前选中文本
- `Scientific Translator: Translate clipboard`：翻译剪贴板内容

---

## 命令面板命令

按 `Ctrl/Cmd + P` 打开命令面板，输入以下关键词：

| 关键词 | 命令 |
|---|---|
| Translate | 翻译选中文本（Scientific Translator） |
| clipboard | 翻译剪贴板内容（Scientific Translator） |

---

## 弹窗内操作

翻译弹窗打开后：

| 操作 | 说明 |
|---|---|
| 点击 🔊 原文 | 用 TTS 朗读原文 |
| 点击 🔊 译文 | 用 TTS 朗读中文译文 |
| 点击 📋 复制 | 复制译文到剪贴板 |
| 点击 × | 关闭弹窗 |
| 按 Esc | 关闭弹窗 |
| 点击弹窗外部 | 关闭弹窗 |
| 按住标题栏拖动 | 移动弹窗位置 |

---

## 鼠标操作

| 操作 | 触发条件 | 效果 |
|---|---|---|
| 选中文字 + 右键 | **markdown 编辑器**里选中 | 弹出右键菜单，含「🔬 科研翻译」 |
| 选中文字 + Ctrl+C | **任何地方**（含 PDF） | 弹窗自动出现 |

> ⚠️ PDF 视图里**右键菜单只有 3 项**（Obsidian 系统默认：复制 / 复制为引用 / 复制到选区的链接），用「复制即翻译」流程更方便。

---

## 推荐工作流

### 读 PDF 论文（推荐 ⭐）

1. 在 PDF 里选中一段英文
2. 按 `Ctrl+C`
3. 弹窗自动出现（带译文 + 音标 + 术语备注）
4. 不懂发音？点 🔊 原文
5. 想把术语粘到笔记？点 📋 复制
6. 关闭弹窗（Esc / 点击外部）

### 读 markdown 笔记

1. 选中一段英文
2. `Ctrl/Cmd + Shift + T`（快捷键）OR `Ctrl+C`（自动）
3. 弹窗显示
4. 点 📋 复制
5. 粘贴到笔记

### 关闭自动翻译（如果你不喜欢）

Settings → Scientific Translator → 「复制后自动翻译」→ 关掉
然后用命令面板手动翻译。
