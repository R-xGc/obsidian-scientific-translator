# 快捷键说明

## 默认快捷键

| 快捷键 | 操作 | 说明 |
|---|---|---|
| `Ctrl/Cmd + Shift + T` | 翻译当前选中文本 | 弹窗显示结果 |
| `Ctrl/Cmd + P` | 打开命令面板 | 输入「Translate」搜索命令 |
| `Esc` | 关闭翻译弹窗 | 在弹窗打开时按 |

---

## 自定义快捷键

Obsidian → Settings → Hotkeys → 搜「Scientific Translator」

可以重新绑定：
- `Scientific Translator: Translate selection`：翻译当前选中文本
- `Scientific Translator: Translate clipboard`：翻译剪贴板内容

---

## 命令面板命令

按 `Ctrl/Cmd + P` 打开命令面板，输入以下关键词：

| 关键词 | 命令 |
|---|---|
| Translate | 翻译选中文本（Scientific Translator） |
| clipboard | 翻译剪贴板内容（Scientific Translator） |

---

## 弹窗内操作

翻译弹窗打开后：

| 操作 | 说明 |
|---|---|
| 点击 🔊 原文 | 用 TTS 朗读原文 |
| 点击 🔊 译文 | 用 TTS 朗读中文译文 |
| 点击 📋 复制 | 复制译文到剪贴板 |
| 点击 × | 关闭弹窗 |
| 按 Esc | 关闭弹窗 |

---

## 鼠标操作

| 操作 | 触发条件 | 效果 |
|---|---|---|
| 选中文字 + 右键 | 编辑器里选中 | 弹出右键菜单，含「🔬 科研翻译」 |

---

## 推荐工作流

### 读英文论文时

1. 选中一段英文
2. `Ctrl/Cmd + Shift + T`
3. 弹窗里看译文和术语备注
4. 不懂发音？点 🔊 原文 听一遍
5. 需要粘贴译文到笔记？点 📋 复制
6. 关闭弹窗（Esc）

### 整理笔记时

1. 选中英文术语
2. `Ctrl/Cmd + Shift + T`
3. 复制术语备注（不只译文）
4. 粘贴到笔记对应位置
